Among all block programming languages is there one with the same purpose than ours?

Accordign to my reseach so far, there is not one with the exactly same purpose than ours but the closest ones to ours are Scratch, Alice,and the websites tutorials; code.org, Khan Academy, and, CS Unplugged.

Similarities to our programming language

  • Scratch

    • Main topics cover both specific operations and project-based instruction with a primary focus on how to use the operations.

  • Alice

    • It is a 3D programming environment focused on creating animations and story telling.It is focused on fundamental programming concepts.

The folloign article is a very nice article about these four virtual education environments

Empowering K–12 Students With Disabilities to Learn Computational Thinking and Computer Programming by Ismael Maya and others. Prnted in 2015.

Moreover, there are very interesting articles that claim that the use of these enviroments in education improve the leanring of students with and without disabilities. For example, the article called Activities-Oriented Science Instruction for students with disabilities repots that in several investignation in scinece education, students with disabiliites overwhelmingly reported enjoying the activities-oriented instruction more than textbook instruction

Activities-Oriented Science Instruction for Students with Disabilities by Margo A Mastropieri and others. Painted in 1999.

The purpose of our language includes programming thinking. so how other block languages are currently doing this?

I found several articles that compare the effectiveness of certain block languages out there when it comes to Education in programming thinkg. For insntace. the article, Comparing Elementary Students' Programming Success based on Programming Environment. shows an study that compares attitudinal and learning outcomes of 7thgrade students programming in Logo and Scratch. And, Results showes that learning Scratch first can provide a better understanding of basic programming concept for novices in elementary school than Logo

Comparing Elementary Students' Programming Success based on Programming Environment. by Monika Miladenovi, and others. Prnted in 2016.
